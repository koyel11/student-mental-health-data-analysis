# Student Mental Health Data Analysis

## Overview
This project analyzes factors affecting student mental health using survey-based data.
The goal is to identify patterns related to stress, sleep, academic pressure, and lifestyle.

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib / Seaborn
- Jupyter Notebook

## Key Insights
- Higher academic workload correlates with increased stress levels
- Poor sleep patterns are strongly associated with anxiety
- Social support shows a protective effect on mental health

## Data
The dataset has been anonymized to protect participant privacy.

## Project Status
Completed exploratory data analysis. Future work may include predictive modeling.
